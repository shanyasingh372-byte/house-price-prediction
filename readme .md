House Price Prediction using Machine Learning

# Description
This project is a Machine Learning application that predicts house prices based on the size of the house. It uses Linear Regression, a supervised learning algorithm, to learn the relationship between house size and price.

#Objective
To build a model that can predict house prices using house size as input.

# Technologies Used
- Python 3.x
- Pandas
- Scikit-learn

# Project Structure
- main.py → Main Python file to run the project
- data.csv → Dataset containing house size and price
- requirements.txt → Required libraries
- README.md → Project documentation

# Environment Setup

1. Install Python (if not installed)
2. Install required libraries using the following command:

pip install -r requirements.txt

# How to Run the Project

1. Open terminal / command prompt
2. Navigate to project folder
3. Run the following command:

python main.py

#Input and Output

- Input: House size (entered by user)
- Output: Predicted house price

Example:
Enter house size: 1000  
Output: Predicted Price: 2500000 (approx)

# Concepts Used
- Machine Learning
- Linear Regression
- Supervised Learning
